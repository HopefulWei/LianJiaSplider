# LianJiaSplider
Statistics of urban house price information on LianJia。

该代码主要用于爬取链家网的城市所有区县的房价数据，并保存在csv文件中，数据的维度如下图，同时可以统计二手房均价。

通过输入想要查询的城市，获取所在城市的信息。
